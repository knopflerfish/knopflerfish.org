This is the subversion repository for
Knopflerfish OSGi.

  http://www.knopflerfish.org

   trunk     -   main development trunk
   branches  -   misc development branches
   tags      -   tagged versions
   releases  -   releases

To check out, use a Subversion client
  http://subversion.tigris.org

A nice windows client is avaiable at
  http://tortoisesvn.tigris.org


typically you can check out the main trunk by:

 > mkdir knopflerfish.org
 > svn co https://www.knopflerfish.org/svn/knopflerfish.org/trunk


If you want to commit to knopflerfish.org, please contact

 info@knopflerfish.org
