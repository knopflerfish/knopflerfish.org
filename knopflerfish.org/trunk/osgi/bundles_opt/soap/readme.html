Knopflerfish and SOAP

There are two different SOAP implementations for Knopflerfish, the Axis one and the kSOAP one.

Axis

The Knopflerfish Axis port is the oldest and largest one. The following subdirectories are part of it:

  • axis-osgi
  • demos
  • soap_desktop
  • soap_remotefw
  • soapclient
  • soapobject

More information is available here: axis.html

kSOAP

The kSOAP implementation is smaller than Axis. Only the following subdirectories are part of it:

  • ksoap-osgi
  • ksoap_remotefw

More information is available here: http://knopflerfish.org/remote_howto.html

KF

Full information on the knopflerfish project can be found at http://www.knopflerfish.org.
