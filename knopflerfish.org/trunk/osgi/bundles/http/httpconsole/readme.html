Knopflerfish HTTP/HTML graphical console


HTTP console on pocketPC 2003

The Knopflerfish HTTP/HTML OSGi console is a small servlet that can be run on any OSGi platform with an installed HTTP server.

When started, the servlet registers at /servlet/console and allows access to the common framework functions as bundle installation, start, stop, update and uninstall. Additionally, detail info on a bundle such as state and manifest info can be displayed.

The HTTP console has been tested on Knopflerfish and Oscar. It also runs well on Pocket PC (tested using KF on the IBM J9 VM).

The main screen lists all installed bundles where one or more bundles can be selected by marking its checkbox on the left. After selecting bundles, the operations available as toolbar icons can ber performed.

A pre-compiled bundle jar can be found at http://www.knopflerfish.org/repo/jars/httpconsole/httpconsole-1.0.0.jar, as well in the KF bundle repository


Toolbar

The toolbar display a set of command icons. When an icon is selected, the command will be performed. Any command result or errors will be displayed to the right of the bundle list.
