Welcome to the Knopflerfish OSGi Service Platform

If you see this page you have successfully started the Knopflerfish OSGi Service Platform.

Knopflerfish OSGi includes an small foot-print HTTP server and this page is published by the httproot demo bundle as an example on how to use the HTTP serviceserver and corresponding OSGi HTTPService.

Content can be served either as static resources (html pages and images), or as dynamic resources from servlets.

This page is a static html page. The HTTP Server info is generated by a servlet in the httproot demo bundle.
