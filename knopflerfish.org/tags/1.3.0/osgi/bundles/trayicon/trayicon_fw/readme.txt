
Simple trayicon supprting start level change and framework shutdown.


System properties for configuration:

org.knopflerfish.service.trayicon.fw.title

  Default is "Knopflerfish OSGi"

org.knopflerfish.service.trayicon.fw.initmsg
  Default is "The Knopflerfish OSGi framework is initialized"

org.knopflerfish.service.trayicon.fw.restartmsg

  Default is "The Knopflerfish OSGi framework is restarted"
