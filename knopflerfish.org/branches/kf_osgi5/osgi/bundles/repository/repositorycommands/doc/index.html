Repository Commands

The Knopflerfish repository manager console bundle

Description

The Repository commands bundle includes a set of commands through which the Knopflerfish Repository Manager can be controlled. This includes bundle search and install, repository management (add, remove, ordering, enable and disable).

Repository Command Group

The console command group is repository

The available commands are: 

Available repository commands:
  add [-help] [-r #rank#] <url> - Add a XML based repository.
  bundle [-help] [-l] [<symbolicname> [<versionRange>]] - Find bundle resources all ot all that matches <symbolicname>
  disable [-help] <repository> ... - Disable repository, so that it won't be used when searching for resources
  enable [-help] <repository> ... - Enable repository, so they will be used when searching for resources
  install [-help] [-s] <symbolicname> [<versionRange>] - Install bundle resource that matches <symbolicname>
  list [-help] [-l] [<repository>] - List repositories
