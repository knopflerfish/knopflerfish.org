About Knopflerfish 4 - an OSGi R4 v4.3 distribution

Knopflerfish 4 (KF4) is the first beta release of the next major release of Knopflerfish. KF4 is an implementation of the OSGi R4 V4.3 specifications. The distribution includes the following components:
  • Knopflerfish OSGi framework - OSGi R4 v 4.3 Core Specification implementation
  • Knopflerfish OSGi bundles - OSGi R4 v 4.3 Service Compendium implementation
  • Knopflerfish components & extras - Knopflerfish bundles and utilities

Knopflerfish 4 OSGi Framework - OSGi R4 v4.3 Core

The Knopflerfish 4 OSGi framework is implemented in accordance with the OSGi R4 v4.3 specification.

The numbering for the OSGi defined components refer to the OSGi specification chapter numbering.

Knopflerfish 4 Components - Bundles and Utilities

The Knopflerfish distribution includes a set of Knopflerfish specific bundles and Utilities. Some of the components include, or are based on other popular open source components / projects. Knopflerfish has in this case typically provided a wrapper class, bundlifying the component / project.
